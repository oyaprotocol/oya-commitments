"Test ERC1155 token" means exactly 1 unit of token id <TEST_ERC1155_TOKEN_ID> from <TEST_ERC1155_TOKEN_ADDRESS>.

The price of 1 Test ERC1155 token is exactly 1 USDC.

The authorized agent may send 1 Test ERC1155 token from the agent wallet to a recipient.
After doing so, the agent may transfer exactly 1 USDC from this Safe to the agent address.

This is allowed only if either:
- the recipient paid exactly 1 USDC into this Safe for that token; or
- the recipient was named in a signed message requesting fast withdrawal of that token.

Each payment or signed message may be used at most once.
